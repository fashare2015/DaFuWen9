# 说明
大家好，我两周前入坑大富翁9。说起来这个游戏真是有毒，头两天一直通宵了刷比赛。终于熬不住了，抽空撸了一款挂机脚本。。。

该脚本基于**按键精灵**，仅支持安卓版
- 需要`root`权限，推荐用`KingRoot`
- `安卓6.0`的手机，需要额外开启该脚本的悬浮窗权限，否则闪退。

![](open.gif)

# 特性
- 自动挂比赛
  - 学徒赛
  - 标准赛
  - 精英赛（开发中）
  - 大师赛（开发中）

![](auto_vs.gif)

- 自动摇金币

![](auto_money.gif)
